#Simple Shell

UNIX command interpreter

##Description

A simple shell that return output accoding to the user input

## Installing

* Compiling the program: `gcc -Wall -Werror -Wextra -pedantic *.c -o hsh`

* Example:
```
$ gcc -Wall -Werror -Wextra -pedantic *.c -o hsh
$ ./hsh
~$ ls
~$ hsh main.c shell.c
~$
~$exit
$
```
